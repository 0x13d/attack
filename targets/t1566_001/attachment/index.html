Benign T1566.001 target — Spearphishing Attachment

This stands in for a phishing page (webmail / file-share) serving an attachment lure. The download below is a harmless text file named like a macro-enabled document. Because this page is served over plain http://localhost, the download's referrer is an insecure-HTTP origin — a delivery-context red flag — so the attack EDR raises a T1566.001 detection. (In the wild the referrer would be a webmail/file-share host.)

⬇ Download invoice.docm
Macro-enabled-document lure, delivered from this (insecure-HTTP) page.

The rule is keyed on the delivery context: the same .docm downloaded directly (no referrer) would not be flagged as spearphishing.
