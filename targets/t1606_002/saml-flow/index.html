Benign T1606.002 target — Forge Web Credentials: SAML

These links navigate to URLs carrying a SAML assertion indicator on a host that is not in the operator's samlTrustedHosts allowlist — the shape of a forged/golden-SAML assertion delivered to an unexpected service provider. With the attack EDR loaded each raises a T1606.002 detection (visibility, detect + alert). No real assertion; nothing runs.
