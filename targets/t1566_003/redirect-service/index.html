Benign T1566.003 target — Spearphishing via Service

This page stands in for a trusted service with an open-redirect. The link below navigates to a URL on this host carrying a ?url= parameter that points off-domain — the laundered-link delivery vector. With the attack EDR loaded, navigating it raises a T1566.003 detection. Nothing actually redirects; the detection is on the URL itself.
